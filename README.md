# Truth-Guardian-VAK
**Truth Guardian** by **Team Vakratunda** (SIT Valachil) is a smart security shield that detects scam intent using **Behavioral DNA**. It uses a **Trident Handshake** to verify if a bank OTP is real before the user even sees it. If a scam is caught, the *Hunter-Protocol* feeds the attacker a **Honey-Token** to trace their physical IP and location.
